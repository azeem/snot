# Project

This is a note taking tool with a chat like interface. The user interacts with app by typing commands or note text into the chat text box.

# Tech Stack

Pure javascript in a single index.html. UI is written using mithriljs. The entire source code is in the single index.html file. Uses pico css framework for minimal CSS. Project doesnt use any image assets ir any other assets that arent otherwise embedded in the index.html file.

# Package Manager

No package managers. Entire source code is in a single index.html file. All dependencies are inlined via cdn links.
